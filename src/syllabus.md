# Course Syllabus - Introduction to Web Development

![Course Logo](./images/logo.png)

## Course Overview

Welcome to Introduction to Web Development! This course will teach you the fundamentals 
of building modern websites.

**Course Description:**

This course covers HTML, CSS, JavaScript, and basic web hosting. Students will learn 
to create responsive, accessible websites through hands-on projects.

**Prerequisites:** None

**Duration:** 8 weeks

**Time Commitment:** 3-5 hours per week

---

## Course Objectives

By the end of this course, you will be able to:

1. Write semantic HTML5 markup
2. Style web pages with modern CSS
3. Add interactivity with JavaScript
4. Deploy websites to the web
5. Follow web accessibility best practices

---

## Required Materials

### Software (All Free)
- Text editor (VS Code recommended)
- Web browser (Chrome or Firefox)
- Git for version control

### Accounts Needed
- GitHub account (free)
- Netlify or similar hosting (free tier)

## Course Schedule

| Week | Topic | Activities |

| 1 | HTML Basics | Build your first page |

| 3 | CSS Layouts | Responsive design |
| 4 | JavaScript Intro | Add interactivity |

## Weekly Structure

Each week includes:

- **Reading/Video** (1-2 hours)
- **Hands-on Lab** (2 hours)
- **Assignment** (1-2 hours)
- **Optional Challenge** (extra practice)

## Tips for Success

1. Practice coding every day, even if just 20 minutes
2. Don't copy/paste - type the code yourself
3. Break big problems into small steps
4. Ask questions early and often
5. Build real projects that interest you
