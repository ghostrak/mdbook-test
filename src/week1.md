# Week 1: Introduction to HTML

## Overview

This week you'll learn the basics of HTML and create your first web page.

## Learning Objectives

By the end of this week, you will:
- Understand what HTML is and how it works
- Know the structure of an HTML document
- Use common HTML tags
- Create a simple web page

## Topics Covered

### 1. What is HTML?
HTML (HyperText Markup Language) is the standard language for creating web pages.

### 2. HTML Document Structure

Every HTML page has this basic structure:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
```

### 3. Common HTML Tags

- `<h1>` to `<h6>` - Headings
- `<p>` - Paragraphs
- `<a>` - Links
- `<img>` - Images
- `<ul>`, `<ol>`, `<li>` - Lists
